Examples
--------

.. code-block:: yaml

    - name: Reboot master nodes
      network.offline_migration_sdn_to_ovnk.reboot_nodes:
        role: "master"
        namespace: "openshift-machine-config-operator"
        daemonset_label: "machine-config-daemon"
        delay: 1
        retries: 5
        retry_delay: 3
        timeout: 1800
